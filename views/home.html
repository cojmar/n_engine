Default Home Page
Useful stuff:
  • cfg/config.php -> you can set default homepage there
  • views/layout.html -> layout. Any <!--content--> will be replaced with page content.
    Example: http:{$this->cfg('path','absolute_path')}home
    will populate the content comment in layout with the content form views/home.html
  • for more examples pls visit: http:{$this->cfg('path','absolute_path')}test
